# <a name="macos-device-settings-to-allow-or-restrict-features-using-intune"></a>Intune을 사용하여 기능을 허용하거나 제한하는 macOS 디바이스 설정

[!INCLUDE [azure_portal](./includes/azure_portal.md)]

이 문서에서는 macOS 디바이스에서 제어할 수 있는 다양한 설정을 나열하고 설명합니다. MDM(모바일 디바이스 관리) 솔루션의 일부로 이러한 설정을 사용하여 기능을 허용하거나 사용하지 않도록 설정하고, 암호 규칙을 설정하고, 특정 앱을 허용 또는 제한합니다.

이러한 설정은 Intune에서 디바이스 구성 프로필에 추가된 다음, macOS 디바이스에 할당되거나 배포됩니다.

## <a name="before-you-begin"></a>시작하기 전에

[장치 제한 구성 프로필을 만들려면](device-restrictions-configure.md#create-the-profile)합니다.

## <a name="general"></a>일반

- **정의 조회 차단**: **차단**을 선택하면 사용자가 단어를 강조 표시한 다음, 디바이스에서 해당 정의를 조회할 수 없습니다. **구성되지 않음**(기본값)을 통해 정의 조회 기능에 액세스할 수 있습니다.
- **받아쓰기 차단**: **차단**을 선택하면 사용자가 음성 입력을 사용하여 텍스트를 입력할 수 없습니다. **구성되지 않음**(기본값)을 통해 사용자가 받아쓰기 입력을 사용할 수 있습니다.
- **차단 콘텐츠 캐싱**: 선택 **구성 되지 않은** 콘텐츠 캐싱을 사용 하도록 설정 (기본값). 콘텐츠 캐싱 응용 프로그램 데이터, 웹 브라우저 데이터, 다운로드 및 장치에서 더 로컬로 저장합니다. 선택 **블록** 이 데이터 캐시에 저장 하지 못하도록 합니다.

  MacOS에서 콘텐츠 캐시에 대 한 자세한 내용은 참조 하세요. [Mac에서 콘텐츠 캐싱 관리](https://support.apple.com/guide/mac-help/manage-content-caching-on-mac-mchl3b6c3720/mac) (다른 웹 사이트 열기).

  이 기능은 다음에 적용됩니다.  
  - macOS 10.13 이상

- **(감독 모드인 경우에) 소프트웨어 업데이트를 연기할**:로 설정 하면 **구성 되지 않은** (기본값), Apple을 릴리스 하는 대로 장치의 소프트웨어 업데이트가 표시 됩니다. 예를 들어, 특정 날짜에 apple macOS 업데이트를 해제 하는 경우 업데이트를 자연스럽 게 표시 됩니다 릴리스 날짜 까지의 장치에서.

  **사용 하도록 설정** 0 ~ 90 일 동안에서 장치에 표시 된 소프트웨어 업데이트를 지연할 수 있습니다. 이 설정은 업데이트 기능은 설치 되지 경우 제어 하지 않습니다. 

  - **소프트웨어 업데이트의 표시 여부를 지연**: 0 ~ 90 일의 값을 입력 합니다. 지연이 만료되면 사용자는 지연이 트리거되었을 때 사용 가능한 가장 오래된 버전의 OS로 업데이트하라는 알림을 받습니다.

    예를 들어, macOS 업데이트에서 사용할 수 있는 경우 **1 월 1 일**, 및 **가시성 지연** 로 설정 되어 **5 일**, 장치에서 사용 가능한 업데이트로 업데이트 표시 되지 않습니다. 에 **6 번째 날** 릴리스, 업데이트를 사용할 수 있고 최종 사용자가 설치할 수 있습니다.

    이 기능은 다음에 적용됩니다.  
    - macOS 10.13.4 이상

## <a name="password"></a>암호

- **암호**: **필요**를 선택하면 최종 사용자에게 디바이스에 액세스하기 위해 암호를 입력해야 합니다. **구성 되지 않은** (기본값)에 암호가 필요 하 고 제한이 같은 단순 암호 차단 하거나 최소 길이 설정 하도록 강제 하지 않습니다.
  - **필수 암호 유형**: 암호를 숫자만으로 구성할 수 있는지 아니면 영숫자(문자와 숫자 포함)로 구성해야 하는지 지정합니다. 이 설정은 Mac OS X 버전 10.10.3 이상에서만 지원됩니다.
  - **암호에 포함해야 하는 영숫자가 아닌 문자 수**: 암호에 필요한 복잡한 문자 수(**0**~**4**)를 지정합니다.<br>복합 문자는 "**?**" 등의 기호입니다.
  - **최소 암호 길이**: 사용자가 구성해야 하는 암호의 최소 길이(**4**~**16**자)를 입력합니다.
  - **단순 암호**: **0000** 또는 **1234**와 같은 단순 암호 사용을 허용합니다.
  - **화면 잠금 후 암호를 요구하기 전까지 최대 시간(분)**: 컴퓨터 잠금을 해제하려면 암호를 입력해야 할 때까지 컴퓨터가 비활성 상태로 유지되는 시간을 지정합니다.
  - **화면이 잠기기 전까지 최대 비활성 시간(분)**: 화면이 잠기기 전까지 컴퓨터를 유휴 상태로 유지해야 하는 기간을 지정합니다.
  - **암호 만료(일)**: 사용자가 암호를 변경해야 할 때까지의 기간(**1**~**255**일)을 지정합니다.
  - **이전 암호 다시 사용 안 함**: 이전에 사용했으며 다시 사용할 수 없는 암호의 수를 **1**~**24**개 사이에서 입력합니다.

- **암호 수정에서 사용자를 차단**: 선택 **블록** 암호 변경, 추가 또는 제거를 중지 합니다. **구성되지 않음**(기본값)을 통해 암호를 추가, 변경 또는 제거할 수 있습니다.
- **지문 잠금 해제 차단**: **차단**을 선택하면 지문으로 디바이스 잠금을 해제할 수 없습니다. **구성되지 않음**(기본값)을 통해 사용자가 지문을 사용하여 디바이스 잠금을 해제할 수 있습니다.

- **암호 자동 채우기 차단**: **차단**을 선택하면 macOS에서 암호 자동 채우기 기능 사용이 제한됩니다. 선택 **블록** 역시 같은 영향을 줍니다.

  - 사용자에게 Safari 또는 앱에서 저장된 암호를 사용할지 묻는 메시지가 표시되지 않습니다.
  - 자동 강력한 암호가 사용되지 않고 강력한 암호가 사용자에게 제안되지 않습니다.

  **구성되지 않음**(기본값)은 이러한 기능을 허용합니다.

- **암호 근접 요청 차단**: **차단**을 선택하면 사용자의 디바이스가 가까운 디바이스의 암호를 요청하지 않습니다. **구성되지 않음**(기본값)은 이러한 암호 요청을 허용합니다.

- **암호 공유 차단**: **차단**을 선택하면 AirDrop을 사용하여 디바이스 간에 암호를 공유할 수 없습니다. **구성되지 않음**(기본값)을 통해 암호를 공유할 수 있습니다.

## <a name="built-in-apps"></a>기본 제공 앱

- **Safari 자동 채우기 차단**: **차단**을 선택하면 디바이스의 Safari에서 자동 채우기 기능을 사용할 수 없습니다. **구성되지 않음**(기본값)을 통해 사용자가 웹 브라우저에서 자동 완성 설정을 변경할 수 있습니다.
- **카메라 차단**: **차단**을 선택하면 디바이스에서 카메라에 액세스할 수 없습니다. **구성되지 않음**(기본값)을 통해 디바이스의 카메라에 액세스할 수 있습니다.
- **Apple Music 차단**: **차단**을 선택하면 Music 앱이 클래식 모드로 되돌아가고 Music 서비스가 사용하지 않도록 설정됩니다. **구성되지 않음**(기본값)을 통해 Apple Music 앱을 사용할 수 있습니다.
- **블록 Spotlight 인터넷 검색 결과**: **블록** 인터넷 검색에서 결과 반환 하는에서 스포트라이트를 중지 합니다. **구성되지 않음**(기본값)을 통해 스포트라이트 검색에서 인터넷에 연결하여 검색 결과를 제공할 수 있습니다.
- **ITunes를 사용 하 여 블록 파일 전송**: **블록** 공유 서비스 응용 프로그램 파일을 사용 하지 않도록 설정 합니다. MacOS 10.13(high에서에서 사용할 수 있는 이상. **구성 되지 않은** (기본값) 응용 프로그램 파일 서비스 공유를 허용 합니다.

## <a name="restricted-apps"></a>제한된 앱

제한된 앱 목록에서 다음 목록 중 하나를 구성할 수 있습니다.

- **금지된 앱** 목록: 사용자가 설치 및 실행할 수 없는 Intune에서 관리되지 않는 앱을 나열합니다. 사용자가 금지 된 앱을 설치할 수 없게 되지 있지만 그럴 경우 관리자에 게 보고 됩니다.
- **승인된 앱** 목록: 사용자가 설치할 수 있는 앱을 나열합니다. 사용자는 나열되지 않은 앱을 설치해서는 안 됩니다. Intune에서 관리되는 앱은 자동으로 허용됩니다. 사용자가 승인된 목록에 없는 앱을 설치할 수 없습니다 되지 않습니다. 하지만 그럴 경우 관리자에 게 보고 됩니다.

목록을 구성하려면 **추가**를 클릭한 다음 선택한 이름, 앱 게시자(선택 사항) 및 앱의 번들 ID를 지정합니다(예: *com.apple.calculator*).

## <a name="connected-devices"></a>연결된 디바이스

- **AirDrop 차단**: **차단**을 선택하면 디바이스에서 AirDrop을 사용할 수 없습니다. **구성되지 않음**(기본값)을 통해 AirDrop 기능을 사용하여 주변 디바이스와 콘텐츠를 교환할 수 있습니다.
- **블록 Apple Watch 자동 잠금 해제**: **블록** 사용자 자신의 Apple Watch 사용 하 여 macOS 장치를 잠금 해제 하는 것을 금지 합니다. **구성 되지 않은** (기본값) 사용자가 해당 Apple Watch 사용 하 여 macOS 장치 잠금을 해제할 수 있습니다.

## <a name="cloud-and-storage"></a>클라우드 및 스토리지

- **iCloud 키 집합 동기화 차단**: **차단**을 선택하면 키 집합에 저장된 자격 증명을 iCloud에 동기화할 수 없습니다. **구성되지 않음**(기본값)을 통해 사용자가 이러한 자격 증명을 동기화할 수 있습니다.
- **Icloud와 문서 동기화 차단**: **블록** iCloud 문서 및 데이터를 동기화 하는 것을 금지 합니다. **구성되지 않음**(기본값)을 통해 iCloud 스토리지 공간에 대한 문서 및 키 값 동기화가 가능합니다.
- **ICloud 백업 메일을 차단**: **블록** iCloud macOS 메일 앱으로 동기화 하는 것을 금지 합니다. **구성 되지 않은** (기본값)를 icloud와 메일 동기화를 허용 합니다.
- **ICloud 연락처 백업 차단**: **블록** iCloud 장치 연락처를 동기화 하는 것을 금지 합니다. **구성 되지 않은** (기본값) iCloud를 사용 하 여 연락처 동기화를 허용 합니다.
- **ICloud 백업 일정을 차단**: **블록** iCloud macOS 일정 앱을 동기화 하는 것을 금지 합니다. **구성 되지 않은** (기본값)를 icloud와 일정 동기화를 허용 합니다.
- **ICloud 백업 알림 차단**: **블록** iCloud macOS 미리 알림 앱에 동기화 하는 것을 금지 합니다. **구성 되지 않은** (기본값) iCloud에 미리 알림을 동기화를 허용 합니다.
- **ICloud 책갈피 백업 차단**: **블록** iCloud 책갈피 장치를 동기화 하는 것을 금지 합니다. **구성 되지 않은** (기본값)를 icloud와 책갈피 동기화를 허용 합니다.
- **ICloud 백업 정보 차단**: **블록** iCloud 장치 정보를 동기화 하는 것을 금지 합니다. **구성 되지 않은** (기본값)를 icloud와 정보 동기화를 허용 합니다.

## <a name="domains"></a>Domains

- **이메일 도메인 URL**: 목록에 하나 이상의 URL을 추가합니다. 사용자가 구성한 도메인이 아닌 다른 도메인에서 받은 이메일은 MacOS 메일 앱에서 신뢰할 수 없는 이메일로 표시됩니다.
