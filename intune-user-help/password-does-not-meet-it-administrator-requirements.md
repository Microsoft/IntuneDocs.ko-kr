# <a name="device-password-requirements-for-enrolled-devices"></a>등록 된 장치에 대 한 장치 암호 요구 사항

조직에서는 회사 또는 학교 리소스에 대 한 액세스를 허용 하기 전에 보다 안전한 암호를 만들어야 할 수 있습니다. 이 문서에서는 Windows 10, iOS, macOS 및 Android 장치에 대 한 일반적인 암호 요구 사항을 설명 합니다. 조직에서 이러한 요구 사항을 모두 적용 하지 않을 수 있습니다.  


암호나 암호가 더 이상 요구 사항을 충족 하지 않는 경우 회사 포털에서 메시지를 받게 됩니다. 이렇게 해야 하는 변경 내용에 대해 설명 합니다. 메시지에서 세부 정보를 제공 하지 않는 경우이 문서를 참조 하 여 현재 암호와 비교할 수 있습니다.  

> [!IMPORTANT]
> 요구 사항에 맞게 암호를 변경 했지만 여전히 알림이 수신 되는 경우 장치를 다시 시작 합니다.  

추가 도움말을 보거나 조직의 특정 요구 사항을 확인 하려면 IT 지원 담당자에 게 문의 하세요. 연락처 정보는 [회사 포털 웹 사이트](https://go.microsoft.com/fwlink/?linkid=2010980)를 참조하세요.  

## <a name="windows-10-password-requirements"></a>Windows 10 암호 요구 사항

| 메시지 | 해결 방법 |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 암호는 필수입니다. | 암호를 설정합니다. 조직에서 장치 잠금을 해제 하려면 암호를 입력 해야 합니다. |
| 암호가 너무 간단합니다. |  암호에는 1234 또는 1111와 같은 순차적 또는 반복적인 숫자가 포함 되지 않아야 합니다. |
| 암호가 너무 짧습니다.| 더 많은 문자를 사용 하 여 암호를 업데이트 하거나 설정 합니다. 조직에서 암호 길이는 일정 해야 합니다. 실제로 선택 하는 것은 다르지만, 필요한 최소 길이는 4 자이 고 최대값은 16입니다. |
| 암호는 숫자만 포함 해야 합니다. | 번호만 포함 하는 암호를 설정 합니다.|
| 암호는 영숫자 문자만 포함 해야 합니다. | 숫자와 문자의 혼합을 포함 하는 암호를 설정 합니다.|
| 암호는 복잡 한 문자를 포함 해야 합니다. | 숫자, 대문자 및 기호 `$`(예:, `%`및 `#`)와 같은 복잡 한 문자를 추가 합니다. 조직에서 문자, 숫자 및 영숫자가 아닌 문자를 함께 사용 하 여 다른 사용자가 암호를 추측 하기 어렵게 만듭니다.|  
| 암호가 만료되었습니다. | 새 암호를 설정 합니다. 조직에서는 특정 일 수 후에 암호를 변경 해야 합니다. |
| 암호가 너무 최근에 사용되었습니다. | 이전에 사용한 적이 없는 암호를 선택합니다. 조직에서는 암호를 다시 사용 하기 전에 특정 시간이 경과 해야 합니다. |

## <a name="ios-passcode-requirements"></a>iOS 암호 요구 사항

| 메시지 | 해결 방법 |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 암호가 필요 합니다.| 암호를 설정합니다. 조직에서 장치를 잠금 해제 하려면 암호를 입력 해야 합니다. |
| 암호가 너무 간단 합니다. |  암호에는 1234 또는 1111와 같은 순차적 또는 반복적인 숫자가 포함 되지 않아야 합니다. |
| 암호가 너무 짧습니다. | 더 많은 문자를 사용 하 여 암호를 업데이트 하거나 설정 합니다. 조직에서 암호의 길이는 일정 해야 합니다. 실제로 선택 하는 것은 다르지만, 필요한 최소 길이는 4 자이 고 최대값은 14입니다. 암호를 변경 하면 Apple에서 6 자 이상의 문자를 입력 하 라는 메시지가 표시 될 수 있습니다. 이 메시지는 Apple 시스템 권장 사항입니다. 조직에서 4 또는 5 자의 암호를 요구 하는 경우 6 자리 암호를 입력할 필요가 없습니다.|  
| 암호는 숫자만 포함 해야 합니다. | 번호만 포함 하는 암호를 설정 합니다.|
| 암호는 영숫자 문자만 포함 해야 합니다.| 숫자와 문자가 혼합 되어 있는 암호를 설정 합니다.|
| 암호에는 영숫자가 아닌 문자를 포함 해야 합니다. | `&` `!` ,,`%`, ,`#`등의 특수 문자를 추가 합니다. `$` 조직에서 문자, 숫자 및 영숫자가 아닌 문자를 함께 사용 하 여 다른 사용자가 암호를 추측 하기 어렵게 만듭니다.|
| 암호가 만료되었습니다. | 새 암호를 설정 합니다. 조직에서는 특정 일 수 후에 암호를 변경 해야 합니다. |
| 암호가 너무 최근에 사용 되었습니다.| 이전에 사용 하지 않은 암호를 선택 합니다. 조직에서는 암호를 다시 사용 하기 전에 특정 시간이 경과 해야 합니다. |
|Touch ID 또는 얼굴 ID 인증이 필요 합니다. | Touch ID 또는 얼굴 ID를 설정 합니다. 조직에서는 암호 또는 신용 카드 정보에 대해 자동 채우기를 사용 하기 전에 이러한 방법 중 하나를 사용 하 여 인증 해야 합니다. |

## <a name="android-password-requirements"></a>Android 암호 요구 사항
| 메시지 | 해결 방법 |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 암호는 필수입니다. | 암호나 PIN을 설정 합니다. 조직에서 장치 잠금을 해제 하려면 암호를 입력 해야 합니다. |
| 암호가 너무 간단합니다. |  암호나 PIN에는 1234 또는 1111와 같은 순차적 또는 반복 되는 숫자가 포함 되지 않아야 합니다. |
| 암호가 너무 짧습니다. | 더 많은 문자를 사용 하 여 암호를 업데이트 하거나 설정 합니다. 조직에서 암호 길이는 일정 해야 합니다.|
| 암호에는 숫자가 포함되어 있어야 합니다. | 숫자가 포함 된 암호 또는 PIN을 설정 합니다.|
| 암호에는 문자가 포함되어 있어야 합니다. | 알파벳 문자를 포함 하는 암호를 설정 합니다.|
| 암호는 영숫자 문자를 포함 해야 합니다. | 숫자와 문자의 혼합을 포함 하는 암호를 설정 합니다.|
| 암호에는 영숫자 문자와 기호가 포함 되어야 합니다. | 문자, 숫자 및 특수 문자 ( `&`예: `!` `$` `%`,,,, `#`)가 혼합 되어 있는 암호를 설정 합니다. |
| 암호는 생체 인식 기술을 사용 해야 합니다.| 지문 또는 얼굴 인식과 같은 생체 인식 인증을 사용 하도록 장치를 설정 합니다.
| 암호가 만료되었습니다. | 새 암호를 설정 합니다. 조직에서는 특정 일 수 후에 암호를 변경 해야 합니다. |
| 암호가 너무 최근에 사용되었습니다. | 이전에 사용한 적이 없는 암호를 선택합니다. 조직에서는 암호를 다시 사용 하기 전에 특정 시간이 경과 해야 합니다. |
